Recuerdas lo que son los paquetes en Python? son mdulos llenos de funciones que te permiten trabajar en tareas ms especficas. Hay uno llamado pandas, el cual es bastante til para manipular datos.

Recuerda, la forma bsica para llamar al paquete de pandas (como a cualquier otro) es:

Esta gua est orientada a los "dataframes", que traducido literalmente puede entenderse como un "encuadrado de datos". stos tienen una estructura elemental:

Para estar familiarizados con el lenguaje, row es fila, y column es columna.

Una consideracin importante al momento de trabajar con paquetes (como pandas), es que una vez cargado el paquete, para cualquier uso posterior ser suficiente el uso directo:

In [ ]:
pandas.DataFrame( data, index, columns, dtype, copy)

Lo que ves all arriba es la forma bsica de cmo crear un dataframe con pandas, donde cada argumento quiere decir lo siguiente:

  • data: los datos pueden tomar varias formas, como arreglos de numpy, series, listas, diccionarios, etc.
  • index: para la etiqueta de las filas, Index es usada para la planilla resultante de datos. La opcin por defecto toma np.arrange(n) si la indexacin no es ingresada.
  • columns: Para las etiquetas de la columna, la sintaxis opcional por defecto es np.arrange(n). Esto es verdadero slo si ninguna indexacin es ingresada.
  • dtype: El tipo de dato para cada columna.
  • copy: este comando es usado para copiar los datos, si la opcin por defecto es False.

Un dataframe de pandas puede ser creado usando varios inputs, tales como

  • Listas
  • diccionarios
  • Series
  • ndarrays de numpy
  • Otro dataframe

A continuacin veremos algunos ejemplos de cmo crear dataframes usando estos inputs.

Crear un dataframe desde un diccionario de ndarrays (arreglos)/listas¶

Todos los ndarrays deben ser del mismo largo. Si la indexacin es ingresada, entonces el largo de los ndices deben ser iguales al largo de los arreglos.

Si la indexacin no es ingresada, entonces por defecto, la indexacin ser range(n), donde n es el largo del arreglo.

Ejemplo 1:

(observe los valores 0,1,2,3. stos son la indexacin default que cumplen con los valores de range(n).

print (df)

1   Jack   34
2  Steve   29
3  Ricky   42

#With two column indices, values same as dictionary keys

second  5  10

first   1 NaN
second  5 NaN

Observa que el data frame df2 es creado con una columna de ndices distinta que la clave del diccionario; entonces, el rellenado de NaNs yace en su lugar. Mientras que df1 es creado con la columna ndices igual a la de las claves del diccionario.

Crear un dataframe desde diccionario de Series¶

El diccionario de Series puede ser ingresado como dataframe. La indexacin resultante es la unin de todas las series indexadas ingresadas.

Ejemplo:

In [14]:
import pandas as pd

d = {'one' : pd.Series([1, 2, 3], index=['a', 'b', 'c']),
   'two' : pd.Series([1, 2, 3, 4], index=['a', 'b', 'c', 'd'])}

df = pd.DataFrame(d)
print (df)

   one  two
a  1.0    1
b  2.0    2
c  3.0    3
d  NaN    4

Nota que para la primera serie, no hay etiqueta "d" ingresada, entonces para la etiqueta "d", se rellena con un NaN.

Agregar, eliminar y seleccionar fila¶

Entenderemos lo siguiente para seleciconar, agregar y eliminar filas a travs de ejemplos. Empecemos con el concepto de la seleccin.

Selección por etiqueta¶

Las filas pueden ser seleccionadas usando la funcin loc e ingresando la etiqueta.

In [24]:
import pandas as pd

d = {'one' : pd.Series([1, 2, 3], index=['a', 'b', 'c']), 
   'two' : pd.Series([1, 2, 3, 4], index=['a', 'b', 'c', 'd'])}

df = pd.DataFrame(d)
print (df.loc['b'])

one    2.0
two    2.0
Name: b, dtype: float64

El resultado es una serie con etiquetas como nombres de columna del dataframe. Y, el nombre de la serie es la etiqueta que se usa para recuperarla.

Eliminación de filas¶

Usa las etiquetas de los ndices para eliminar filas de un dataframe. Si la etiqueta est duplicada, entonces las filas que contengan a esa etiqueta sern borradas.

Si observas, en el ejemplo anterior, las etiquetas estn duplicadas. Borraremos una etiqueta y veremos cmo varias filas son borradas.

In [28]:
import pandas as pd

df = pd.DataFrame([[1, 2], [3, 4]], columns = ['a','b'])
df2 = pd.DataFrame([[5, 6], [7, 8]], columns = ['a','b'])

df = df.append(df2)

# Drop rows with label 0
df = df.drop(0)

print (df)

   a  b
1  3  4
1  7  8

En el ejemplo de arriba, dos filas son borradas porque aquellas contienen la etiqueta 0.
